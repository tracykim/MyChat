客户端发送消息格式定义：

1. 消息类型：
   - 0、添加好友
   - 1、添加群聊
   - 2、创建群聊
   - 3、删除好友
   - 4、删除群聊
   - 5、单聊
   - 6、群聊
2. From用户
3. To用户（或群聊）
4. 真实数据

服务端发送消息格式定义：

1. 消息类型：
   - 0、被添加好友
   - 2、被拉入群聊
   - 3、被删除好友
2. From用户
3. To用户（或群聊）
4. 真实数据

其他类型数据发给客户端，客户端直接显示



## 通信流程

1. 启动服务器
2. 启动客户端
3. 用户登录（没有账号就注册）
4. 查看联系人（包括聊天群）
5. 给联系人发送消息 TODO
6. 退出登录
7. 关闭客户端
